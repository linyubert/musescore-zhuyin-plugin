# MuseScore 注音符號自動標示外掛 (Zhuyin / Bopomofo Plugin)

這是一個專為 MuseScore 設計的外掛，可以一鍵自動讀取樂譜中的音符，並在五線譜下方生成對應的**注音符號 (ㄅㄆㄇ)**。非常適合用於兒童音樂教學、合唱團練習，或是協助初學者快速視譜。

## 📌 功能特色

* **全自動標示：** 只要一鍵點擊，就會自動掃描整份樂譜並標上注音，不需逐字手動輸入。
* **支援首調唱名法：** 程式會自動讀取樂譜的「調號」(Key Signature)，例如遇到 G 大調時，會自動將 G 音標示為「ㄉㄛ」。
* **多聲部/多樂器支援：** 會自動走訪樂譜中的所有五線譜（所有樂器），一次處理完成。
* **自動處理升降音：** 遇到變化音時，會自動加上 `#` 或 `b` 記號（如 `#ㄈㄚ`）。

## 📥 安裝方法

1. 點擊上方綠色的 `Code` 按鈕，選擇 `Download ZIP` 下載檔案，並解壓縮。
2. 將裡面的 `.qml` 檔案複製到 MuseScore 的外掛資料夾中：
   * **Windows:** `C:\Users\你的使用者名稱\Documents\MuseScore3\Plugins` (或 MuseScore4 的對應資料夾)
   * **Mac:** `~/Documents/MuseScore3/Plugins`
3. 重新啟動 MuseScore 軟體。
4. 在上方選單列選擇 **「外掛 (Plugins)」** -> **「外掛管理員 (Plugin Manager)」**。
5. 在清單中找到本外掛，將旁邊的框框**打勾**，按下確定。

## 🚀 使用方式

1. 打開你的 MuseScore 樂譜。
2. 點選上方選單的 **「外掛 (Plugins)」** -> **「自動生成注音(首調)」**。
3. 注音就會瞬間出現在音符下方了！

## 📝 備註
* 如果是遇到和弦（多音疊代），本外掛預設會讀取該和弦的「最高音」來標示注音。
* 生成的注音屬於 MuseScore 的「譜表文字 (Staff Text)」，生成後若有需要，仍可手動點擊文字進行微調。
